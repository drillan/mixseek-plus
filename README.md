# mixseek-plus

mixseek-coreの拡張パッケージ。Groqモデルサポート、Playwrightフェッチ機能強化などの追加機能を提供。

## ドキュメント

- [SPEC.md](./SPEC.md) - プロジェクト仕様書
- [ARCHITECTURE.md](./ARCHITECTURE.md) - アーキテクチャ設計
- [ROADMAP.md](./ROADMAP.md) - 開発ロードマップ

## クイックスタート

```bash
# インストール
pip install mixseek-plus

# 基本的な使用例
mixseek-plus team "タスク" --config team.toml
```

## 主要機能

- mixseek-coreの全機能を継承
- Groqモデルサポート（PlainMemberAgent）
- Playwrightによるweb_fetch強化
- （将来）Claude Codeモデル統合
- （将来）ファイル操作ツール
